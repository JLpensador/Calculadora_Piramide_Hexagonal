# Calculadora_Piramide_Hexagonal
Um projeto de matemática sobre uma calculadora de pirâmide hexagonal <br> onde você calcula a área da base, área lateral, área total e o seu volume <br> Feito com HTML, CSS, Javascript e Python (terminal) <br> <br>
